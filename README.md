# 🚀 NC-Viewer G-Code Visualization

### by **Amir Mobasheraghdam**

## 📌 Overview / Überblick

This project provides a powerful, browser-based **G-Code visualization and parsing system**.
It supports real-time graphic display, tool-path interpretation, and advanced handling of motion commands—
perfect for CNC, 3D-printing, and machining workflows.

> Dieses Projekt bietet ein modernes, browserbasiertes **G-Code-Visualisierungssystem** mit präziser
> Bewegungsauswertung und robustem Parsing-Support für professionelle CNC-Anwendungen.

---

## ✨ Features / Funktionen

✅ Full G-Code parser with support for:

* **G90 / G91** – Absolute / Relative positioning
* **G20 / G21** – Inches / Millimeters
* **G0 / G1** – Linear movements
* **G2 / G3** – Arcs with IJ center
* Arc-to-line segmentation
* Robust canvas resizing using ResizeObserver + DPR correction

> ✅ Unterstützung für vollständigen G-Code-Parser
> inkl. G90/G91, G20/G21, G0/G1, G2/G3 und Segmentierung von Bögen
